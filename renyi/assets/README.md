# assets

- `NanumGothic.ttf` — Korean font for the Phase D figures (`bistatic_scene.make_figures`).
  NanumGothic by Naver, SIL Open Font License 1.1 (redistributable). Bundled so the
  Korean-annotated figures regenerate without a system font; `_setup_style()` falls back
  to English (DejaVu Sans) if it is absent.
